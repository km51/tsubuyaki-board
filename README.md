# つぶやき掲示板  
PHPとMySQLの学習成果物として、Twitterクローンを作成しました。
Udemy講座「PHP+MySQL（MariaDB） Webサーバーサイドプログラミング入門」での作成物を参考に、機能を追加していきます。  
Herokuでデプロイしました。[つぶやき掲示板（Heroku）](https://tsubuyaki-board.herokuapp.com/login.php)
## 基本機能
* 文章の投稿  
* 他の人の投稿に返信  
* ユーザ登録  
* 写真をアップロード  
* ログイン  
* ログアウト  
* ページネーション  
## 環境  
* 言語 : PHP 8.0.0
* DB : MariaDB 10.4.17

## 追加機能
